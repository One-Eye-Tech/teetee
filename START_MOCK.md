# 🚀 快速启动 Mock 模式

## 步骤 1: 停止当前服务器

如果开发服务器正在运行，请先停止它：

- Windows: 按 `Ctrl + C`
- Mac: 按 `Cmd + C`

## 步骤 2: 重新启动

在 `app-web` 目录下运行：

```bash
npm run dev
```

## 步骤 3: 验证

打开浏览器访问 `http://localhost:5173`（或显示的端口）

按 `F12` 打开开发者工具，在控制台中查看：

```
=== 应用配置 ===
USE_MOCK: true    <-- 这里应该是 true
...
```

## 步骤 4: 查看作品

首页应该显示 12 个示例作品，包括：
- 夏日回忆
- 城市印象
- 自然风光
- 等等...

## ✅ 成功标志

- 控制台显示 `USE_MOCK: true`
- 控制台显示 `[worksApi] 使用 Mock 数据获取作品列表`
- 首页显示作品网格
- 没有网络请求错误

## ❌ 如果失败

1. 确认 `.env.development` 文件中有 `VITE_USE_MOCK=true`
2. 确保完全重启了开发服务器（不是热重载）
3. 清除浏览器缓存并硬刷新（Ctrl+Shift+R）
4. 查看控制台错误信息

## 📝 测试功能

现在你可以测试：

- ✅ 浏览作品列表
- ✅ 点击作品查看详情
- ✅ 点赞作品（需要先登录）
- ✅ 用户登录（验证码在控制台）

## 🔄 切换回真实后端

修改 `.env.development`：

```env
VITE_USE_MOCK=false
```

然后重启服务器。
